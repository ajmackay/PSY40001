2.1 Conducting a Multiple Regression

I haven’t yet found a way to have all the regression output as nicely formatted as it appears in SPSS. The closest I could get was to save the regression output in a separate APA formatted document, which I achieved with using the function apa.reg.table located in the apaTables package. The APA output is saved in a folder called “output” which you can access from gitHub if you so desire.

lab2.lm <- lm(anxiety ~ socialid + selfest + raceab + CBT, data = lab2_raw); #Saving Linear Model in lab2.lm object

apa.reg.table(lab2.lm, filename = "output/wk2/Multiple-Regression.doc", table.number = 4)
